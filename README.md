# FoodCall

FoodCall is a food order and delivery application. The interface supports both, the customer and the vendor. 

Customers can perform the following functionalities:
  1. Login
  2. Sign Up
  3. Browse Restaurants
  4. Order Food (multiple items with different quantities supported)
  5. View Order History
  
Vendors can perform the following functionalities:
  1. Login
  2. Sign Up
  3. Add/Edit/Delete Restaurant
  4. Add/Edit/Delete Menu Items
  5. Receive Orders
  6. View Order History
  
The application also supports Notifications through Google Console. The database is hosted on Google Firebase which is a great NoSQL tool by Google to create applications with seamless flow.
